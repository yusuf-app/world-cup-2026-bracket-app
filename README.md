# 2026 World Cup Bracket App

An interactive mobile-first application developed for the FIFA World Cup 2026.

Users can:

- Create group standings
- Select the best third-placed teams
- Generate Round of 32 matchups
- Build the tournament bracket
- Determine the champion
- Export shareable PNG images

## Features

- 48 teams / 12 groups
- FIFA 2026 format
- Best third-place selection
- Interactive bracket builder
- PNG export
- Mobile-friendly design
- Android & iOS compatible architecture
